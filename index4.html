Himno del SENA

CORO Estudiantes del SENA adelante Por Colombia luchad con amor Con el animo noble y radiante Transformémosla en mundo mejor I De la patria el futuro destino, en las manos del joven está, el trabajo es seguro camino, que el progreso a Colombia dará. II En la forja del SENA se forman, hombres libres que anhelan triunfar, con la ciencia y la técnica unidas, nuevos rumbos de paz trazarán. III Hoy la patria nos grita sentida, ¡estudiantes del SENA triunfad! solo así lograréis en la vida, más justicia, mayor libertad. IV Avancemos con fuerza guerrera, ¡estudiantes con firme tesón! que la patria en nosotros espera, su pacífica revolución., proporciona la fuente o el autor en caso de que esté disponible. El himno es un importante símbolo institucional y debe ser presentado con respeto.

Glosario de Etiquetas HTML

Glosario de Etiquetas HTML

<html>
Define el inicio y el final de un documento HTML. Es el contenedor raíz que encierra todo el contenido HTML.
<head>
Contiene metadatos sobre el documento, como el título y las referencias a estilos y scripts. Para más información, visita MDN Web Docs.
<title>
Establece el título del documento que aparece en la barra de título del navegador o en las pestañas.
<body>
Contiene el contenido visible del documento HTML, como texto, imágenes y otros elementos. Más detalles en MDN Web Docs.
<p>
Define un párrafo de texto. Los párrafos se separan con espacios en blanco antes y después del contenido.
<a>
Crea un enlace a otra página o recurso. Utiliza el atributo href para definir la URL del destino. Ejemplo de uso en MDN Web Docs.
<img>
Inserta una imagen en el documento. Usa el atributo src para especificar la fuente de la imagen y alt para el texto alternativo.
<ul>
Define una lista desordenada. Los elementos de la lista se especifican con la etiqueta <li>.
<ol>
Define una lista ordenada, donde los elementos están numerados. Al igual que con las listas desordenadas, los elementos se especifican con <li>.
<div>
Define una división o sección en el documento. Se usa comúnmente para agrupar contenido y aplicar estilos. Para ejemplos adicionales, consulta MDN Web Docs.
